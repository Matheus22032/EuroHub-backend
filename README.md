# EuroHub

Back-end do projeto desenvolvido no challenge da FIAP com Eurofarma

## Instalação

### 1. Clone o repositório

```bash
git clone https://github.com/Matheus22032/EuroHub-backend.git
cd seu-repositorio
```
### 2. Inicialize o Banco de Dados
```bash
docker compose up -d
```
### 2. Inicialize o Banco de Dados
```bash
./mvnw -e spring-boot:run
```
